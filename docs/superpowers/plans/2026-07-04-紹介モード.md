# 紹介モード（PINログイン＝ダミーデータのデモ）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** PINログインで入ると全データがダミーになる「紹介モード」を追加する（実データ・同期を一切汚さない）。

**Architecture:** `sessionStorage.demo_mode==='1'` のとき、メイン`<script>`の先頭で `window.localStorage` をメモリ上の偽ストレージ（ダミーデータ入り）に差し替える。本体コードは無変更で全機能がダミーデータ上で動く。同期は「偽ストアが同期URLを常に空にする」＋「`_gasUrlOne()`のDEMOガード」で二重ブロック。仕様書：`docs/superpowers/specs/2026-07-04-紹介モード-design.md`

**Tech Stack:** プレーンJS（単一HTMLファイル `index.html` にインライン）。検証は node の構文チェック＋ブラウザconsole。

```
<script>
  // ===== 紹介モード（デモ）シム =====
  // PINログインで入ると sessionStorage.demo_mode='1' が立ち、ここで localStorage を
  // メモリ上の偽ストレージ（ダミーデータ入り）に差し替える。実データは読まれも書かれもしない。
  // 同期URL系キーは常に null を返す＝同期は起動しない（_gasUrlOne のガードと二重防御）。
  window.DEMO_MODE = false;
  try { window.DEMO_MODE = (sessionStorage.getItem('demo_mode') === '1'); } catch (e) {}
  if (window.DEMO_MODE) {
    (function installDemoStorage() {
      var BLOCKED = { 'gas_url': 1, 'gas_url_seed': 1, 'gas_url_driving': 1, 'gas_token': 1 };
      var fake = {};
      Object.defineProperty(fake, 'getItem',    { value: function (k) { k = String(k); if (BLOCKED[k]) return null; return Object.prototype.hasOwnProperty.call(this, k) ? String(this[k]) : null; } });
      Object.defineProperty(fake, 'setItem',    { value: function (k, v) { k = String(k); if (BLOCKED[k]) return; this[k] = String(v); } });
      Object.defineProperty(fake, 'removeItem', { value: function (k) { delete this[String(k)]; } });
      Object.defineProperty(fake, 'key',        { value: function (i) { return Object.keys(this)[i] ?? null; } });
      Object.defineProperty(fake, 'clear',      { value: function () { var self = this; Object.keys(self).forEach(function (k) { delete self[k]; }); } });
      Object.defineProperty(fake, 'length',     { get: function () { return Object.keys(this).length; } });
      try {
        Object.defineProperty(window, 'localStorage', { value: fake, configurable: true });
        // セルフテスト：差し替えが効いているか（読み書き・列挙・実ストレージ非汚染）
        localStorage.setItem('__demo_selftest__', 'ok');
        var pass = (localStorage.getItem('__demo_selftest__') === 'ok')
          && (Object.keys(localStorage).indexOf('__demo_selftest__') >= 0)
          && (localStorage.length >= 1);
        localStorage.removeItem('__demo_selftest__');
        if (!pass) throw new Error('demo storage self-test failed');
      } catch (err) {
        // 差し替えに失敗したら紹介モードを開始しない（実データ画面にも進まない）
        try { sessionStorage.removeItem('demo_mode'); } catch (e) {}
        alert('この環境では紹介モードを開始できませんでした。通常のログイン画面に戻ります。');
        location.reload();
        throw err;
      }
      if (typeof window._demoSeed === 'function') window._demoSeed();
    })();
  }

  // localStorage の T26xxx → Z26xxx キー移行
```

```javascript
(function () {
```
