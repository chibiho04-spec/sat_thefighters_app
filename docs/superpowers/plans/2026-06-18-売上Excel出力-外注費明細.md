# 売上表 Excel出力（外注費明細・.xls整形）実装計画

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 売上画面のダウンロードを、外注費を外注先ごとに明細展開し、罫線・列幅・￥書式つきでExcel/Numbers両対応の `.xls`（HTML表）として出力するよう変更する。

**Architecture:** 既存の売上集計 `_calcCaseSales` / `_collectSalesItems` に「外注明細リスト `outsourceItems`」を**非破壊で追加返却**する。外注費の合計値・計算式は一切変えない（明細は内訳として並べるだけ）。出力関数 `_writeSalesCSV` を `_writeSalesExcel`（HTML表→`application/vnd.ms-excel` の.xls、案件情報は `rowspan` で縦結合、金額は `mso-number-format` で￥表示）に置換し、2つの呼び出し元（ヘッダー📥・月別📥）を差し替え、ボタン名を「📥 Excel」に変更。

**Tech Stack:** 単一HTMLファイル（インラインJS）、ブラウザのみ（Blob/`URL.createObjectURL`）、Excel HTML表方式（`mso-number-format`）。テストフレームワークは無いため、検証は `node --check` 相当の構文チェック＋Chrome MCP のブラウザ実機プローブ（金額一致アサーション＋生成HTMLの描画確認）で行う。

**対象ファイル:** `案件管理アプリ/index.html`（このファイルのみ編集）

`function _calcCaseSales(num, master) {`（12774行）の直前に挿入：

```javascript
  // 外注費の明細行＋合計を返す（売上表Excel出力用・計算式は既存 _calcCaseSales と完全一致）。
  // 返り値 items: [{name, cost, isTax?}]（課税アイテムは税抜額・非課税はそのまま・末尾に消費税行）。
  //         cost: Math.round(課税税抜合計×1.1) + 非課税合計（＝既存 outsourceCost と同値）。
  function _collectOutsourceItems(num, lines, status) {
    const rows = [];
    let base = 0, exempt = 0;
    const add = (name, cost, taxable) => {
      cost = Number(cost) || 0; if (!cost) return;
      rows.push({ name: name || '外注', cost });
      if (taxable) base += cost; else exempt += cost;
    };
    try {
      const saved = (typeof loadWS === 'function') ? loadWS(num) : {};
      ((typeof loadOutsourceItems === 'function') ? loadOutsourceItems(saved) : [])
        .forEach(o => add(o.name, o.cost, o.taxable !== false)); // WS外注：旧データ(taxable無)=課税
      add('機材（リース）', String(saved['wsd-rental-cost'] || '').replace(/[^\d]/g, ''), true); // レンタル=課税
    } catch (e) {}
    (lines || []).forEach(l => {
      if (l && l.outsource) add(l.name, (Number(l.qty)||0) * (Number(l.price)||0), l.taxable !== false);
    });
    if (status && Array.isArray(status.outsourceExtraList)) {
      status.outsourceExtraList.forEach(o => add(o.name || '外注費', o.cost, o.taxable === true)); // 外注費リスト：true のみ課税
    } else if (status && Number(status.outsourceExtra)) {
      add('外注費', status.outsourceExtra, false); // 旧・単一値=非課税
    }
    const tax = Math.round(base * 1.1) - base;
    if (tax > 0) rows.push({ name: '消費税', cost: tax, isTax: true });
    return { items: rows, cost: Math.round(base * 1.1) + exempt };
  }
```

## Task 2: 出力を `_writeSalesExcel`（HTML表 .xls）に置換し、呼び出し元を差し替え

**Files:**
- Modify: `案件管理アプリ/index.html`（`_writeSalesCSV` 13489-13520 を置換／`exportSalesCSVForMonth` 13486・`exportSalesCSV` 13535 の呼び出し）

- [ ] **Step 1: `_writeSalesCSV` 関数全体（コメント行13489〜13520）を `_writeSalesExcel` に置換**

```javascript
  // 売上表を Excel整形ファイル(.xls/HTML表) として書き出す共通処理。
  // exportSalesCSV（現フィルタ）/ exportSalesCSVForMonth（月別）両方から使う。
  // 外注費は外注先ごとに行展開（末尾に消費税行）。案件情報は rowspan で縦結合。
  // Excel: 罫線・列幅・￥書式（mso-number-format）／ Numbers: 数値表示。読み取り専用（保存・同期なし）。
  function _writeSalesExcel(filtered, fileSuffix) {
    const esc = (typeof escapeHtml === 'function') ? escapeHtml : (s => String(s == null ? '' : s).replace(/[&<>"]/g, c => ({'&':'&amp;','<':'&lt;','>':'&gt;','"':'&quot;'}[c])));
    const cols = [
      { h:'受注No',   w:62,  num:false }, { h:'日付', w:78, num:false },
      { h:'取引先',   w:130, num:false }, { h:'件名', w:200, num:false },
      { h:'税抜額',   w:78,  num:true  }, { h:'消費税', w:66, num:true },
      { h:'合計',     w:88,  num:true  }, { h:'外注先', w:150, num:false },
      { h:'外注費',   w:78,  num:true  }, { h:'売り上げ', w:88, num:true },
      { h:'ステータス', w:70, num:false },
    ];
    const numCell = (v, attrs) => `<td class="num"${attrs||''}>${Number(v)||0}</td>`;
    const txtCell = (v, attrs, cls) => `<td${cls?' class="'+cls+'"':''}${attrs||''}>${esc(v == null ? '' : v)}</td>`;

    const bodyRows = [];
    filtered.forEach(it => {
      const osItems = (it.outsourceItems && it.outsourceItems.length) ? it.outsourceItems : [{ name:'', cost:0, _blank:true }];
      const n = osItems.length;
      const span = n > 1 ? ` rowspan="${n}"` : '';
      const statusTxt = it.paid ? '入金済' : (it.billed ? '請求済' : '未請求');
      const net = (it.netInhouse != null) ? it.netInhouse : ((it.total||0) - (it.outsourceCost||0));
      osItems.forEach((os, i) => {
        let tds = '';
        if (i === 0) {
          tds += txtCell(it.num, span, 'b') + txtCell(it.date, span) + txtCell(it.client, span) + txtCell(it.title, span);
          tds += numCell(it.subtotal, span) + numCell(it.tax, span) + numCell(it.total, span);
        }
        // 外注先・外注費（明細行ごと）
        if (os._blank) { tds += txtCell('') + numCell(0); }
        else { tds += txtCell(os.name, '', os.isTax ? 'tax' : '') + `<td class="num${os.isTax ? ' tax' : ''}">${Number(os.cost)||0}</td>`; }
        if (i === 0) { tds += numCell(net, span) + txtCell(statusTxt, span); }
        bodyRows.push(`<tr>${tds}</tr>`);
      });
    });

    const sumTotal     = filtered.reduce((s, it) => s + (it.total || 0), 0);
    const sumOutsource = filtered.reduce((s, it) => s + (it.outsourceCost || 0), 0);
    const sumNet       = sumTotal - sumOutsource;
    const sumRow = `<tr class="sum">`
      + `<td colspan="6">合計（請求金額 − 外注費 ＝ 売上）</td>`
      + `<td class="num">${sumTotal}</td><td></td><td class="num">${sumOutsource}</td>`
      + `<td class="num">${sumNet}</td><td></td></tr>`;

    const headRow = '<tr>' + cols.map(c => `<th style="width:${c.w}px;">${esc(c.h)}</th>`).join('') + '</tr>';
    const html =
      '<html xmlns:o="urn:schemas-microsoft-com:office:office" xmlns:x="urn:schemas-microsoft-com:office:excel" xmlns="http://www.w3.org/TR/REC-html40">'
      + '<head><meta http-equiv="Content-Type" content="text/html; charset=UTF-8">'
      + '<style>'
      + 'table{border-collapse:collapse;}'
      + "td,th{border:0.5pt solid #888888;padding:3px 6px;font-size:11pt;font-family:'游ゴシック','Yu Gothic',sans-serif;vertical-align:top;mso-data-placement:same-cell;}"
      + 'th{background:#dfe7ef;font-weight:bold;text-align:center;}'
      + 'td.b{font-weight:bold;}'
      + 'td.tax{color:#666666;font-style:italic;}'
      + 'tr.sum td{background:#eef2f6;font-weight:bold;}'
      + '.num{text-align:right;mso-number-format:"\\¥#\\,##0;\\-\\¥#\\,##0";}'
      + '</style></head><body>'
      + '<table>' + headRow + bodyRows.join('') + sumRow + '</table>'
      + '</body></html>';

    const bom = '﻿';
    const blob = new Blob([bom + html], { type: 'application/vnd.ms-excel;charset=utf-8' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = `売上表_${fileSuffix}_${new Date().toISOString().slice(0,10)}.xls`;
    a.click();
    URL.revokeObjectURL(url);
  }
```
